# orbit
orbit 은 space marketplace app 으로 space 에 예약메시지 기능을 추가하는 project 입니다.

## prerequisites
* node.js 18.16
* yarn

## config
실행을 위해서 .env 설정이 필요합니다.  
자세한 값들은 프로젝트 문서를 확인해주시기 바랍니다. ([문서](https://beyond-imagination.jetbrains.space/documents/R98H81SkA4o))
```shell
cp .env.template .env
```

## Getting Started

```shell
yarn install
yarn dev
```
